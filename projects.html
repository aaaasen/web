RTree Compression Codec

The RTree Compression Codec (.rcc) uses an RTree to convert bitmap images into a rectangle based format that can efficiently compress large images of varying complexity.
ARCSim

ARCSim is a particle simulator written in Java with Java SWT graphics and support for different elements and chemical reactions. It includes a homebrewed 3D renderer.
Recollection

Recollection is a 2D adventure game written in Java with our homebrewed engine. You play as an adventurer, wandering through an errie maze as it changes around you. Recollection was built during a 24 hour hackathon at StudentRND with Dylan Swiggett and Michael Rosenberger.

Space Cat

Space Cat is an Android game built with libgdx in which you pilot a cat through space, trying to make it to your home planet while avoiding enemy planets and black holes. It was built over the course of 31 hours at a StudentRND hackathon with Dylan Swiggett and Michael Rosenberger.
